📈 XIRR Calculator — True Return on SIP / Multiple Investments
XIRR gives you the actual annualised return accounting for when each investment was made. Far more accurate than simple return % for SIPs and multiple purchases.
How to use: Enter each investment date and amount (negative = money invested, positive = current value / redemption). Add a final row with today's date and current portfolio value as a positive number.

💡 XIRR vs Simple Return — What's the Difference?
❌ Simple Return (Misleading)
Invested ₹1,20,000 over 12 months (₹10,000/month)
Current value: ₹1,35,000
Simple return = 12.5%
But your first SIP was invested for 12 months, last SIP for only 1 month!
✅ XIRR (Accurate)
Same investment, same value
XIRR = ~22-24% p.a.
Accounts for timing of each cash flow
This is what your fund statement shows as "absolute return"
